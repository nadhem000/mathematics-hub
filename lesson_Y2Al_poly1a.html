Function Definition

Formal Definition

Let E be a non-empty subset of ‚Ñù. When to each real number x in E, we associate at most one real number y, we say we have defined a function from E to ‚Ñù.

If we denote this function by f, we write y = f(x) and note:

f: E ‚Üí ‚Ñù
x ‚Ü¶ f(x)

The set D of real numbers x in E such that f(x) exists is called the domain of definition of the function f. We then say that f is defined on D.

Vocabulary

Let D be the domain of definition of f, x a real number belonging to D, and y = f(x).

We say that:

  • The real number y is the image of x by the function f.
  • x is an antecedent of the real number y by f.
