Exercise 4: Arithmetic Mean

a) Show that if a, b and c are three consecutive terms of an arithmetic sequence, then b = (a+c)/2. (We say that b is the arithmetic mean of a and c).

b) Let a, b and c be three real numbers such that b = (a+c)/2. Are a, b and c three consecutive terms of an arithmetic sequence?

In an arithmetic sequence, the difference between consecutive terms is constant.

Part a: Proving b = (a+c)/2

Step 1: Understand what "consecutive terms" means

If a, b, c are consecutive terms of an arithmetic sequence, then:

Let r be the common difference

b = a + r

c = b + r = a + 2r

Step 2: Express (a+c)/2 in terms of a and r

Calculate (a+c)/2:

(a+c)/2 = [a + (a + 2r)] / 2

(a+c)/2 = (2a + 2r) / 2

(a+c)/2 = a + r

Step 3: Compare with b

But we know b = a + r

Therefore: b = (a+c)/2

This proves that b is exactly the average of a and c!

Part b: The converse

Step 1: Start with b = (a+c)/2

We have: b = (a+c)/2

Multiply both sides by 2: 2b = a + c

Step 2: Check if there exists a common difference r

For a, b, c to be consecutive terms of an arithmetic sequence, we need:

Let's check if b - a equals c - b:

From 2b = a + c, we get c = 2b - a

Now c - b = (2b - a) - b = b - a

Yes! b - a = c - b

Step 3: Conclusion

Since b - a = c - b, let's call this common value r

Then: b = a + r and c = b + r = a + 2r

So a, b, c are indeed consecutive terms of an arithmetic sequence with common difference r

Answer:

a) Proof shown above: b = (a+c)/2 when a, b, c are consecutive terms of an arithmetic sequence

b) Yes, if b = (a+c)/2, then a, b, c are consecutive terms of an arithmetic sequence

Exercise 5: Syracuse Sequence

Note: This is NOT an arithmetic sequence! It's a special sequence with its own rules.

We define a Syracuse sequence as follows: Choose an integer N (the first term).

  • If N is even, the next term is N/2
  • If N is odd, the next term is 3N + 1

We continue in the same way with the new term. We stop when we reach 1.

Examples: For N=5: 5, 16, 8, 4, 2, 1. For N=24: 24, 12, 6, 3, 10, 5, 16, 8, 4, 2, 1.
