Polynomials: Part 4 - Factorization and Roots

Definition

Let P and Q be two polynomials.

We say that polynomial P is factorable by polynomial Q if there exists a polynomial R such that, for all real x,

\(P(x) = Q(x) \cdot R(x)\)

This means that Q is a factor of P, and we can write P as the product of Q and another polynomial R.

Important Note

If a polynomial P is factorable by (x - a), then a is a root of P.

This means that P(a) = 0, and (x - a) is a factor of P(x).

Activity 16: Exploring Polynomial Factorization

Part 1: Relationship Between Factors and Roots

Let a be a real number.

Prove that if a polynomial is factorable by (x - a), then a is a root of this polynomial.

Step 1: Start with the given information

We know that the polynomial P is factorable by (x - a). This means there exists a polynomial R such that:

\(P(x) = (x - a) \cdot R(x)\)

Step 2: Evaluate P at x = a

To prove that a is a root of P, we need to show that P(a) = 0.

\(P(a) = (a - a) \cdot R(a) = 0 \cdot R(a) = 0\)

Step 3: Conclusion

Since P(a) = 0, a is indeed a root of the polynomial P.

This proves that if a polynomial is factorable by (x - a), then a is a root of that polynomial.

Part 2: Cubic Polynomials and Factorization

Let a be a real number and P be the polynomial defined by

\(P(x) = ax^3 + bx^2 + cx + d\)

where a, b, c and d are real numbers with \(a \neq 0\).

Part 2a: Factor \(P(x) - P(a)\)

Step 1: Write the expression

\(P(x) - P(a) = (ax^3 + bx^2 + cx + d) - (aa^3 + ba^2 + ca + d)\)

Step 2: Simplify the expression

\(P(x) - P(a) = a(x^3 - a^3) + b(x^2 - a^2) + c(x - a)\)

Step 3: Factor each term

\(x^3 - a^3 = (x - a)(x^2 + ax + a^2)\)
\(x^2 - a^2 = (x - a)(x + a)\)
\(x - a = (x - a)\)

Step 4: Combine the factors

\(P(x) - P(a) = (x - a)[a(x^2 + ax + a^2) + b(x + a) + c]\)

Step 5: Final factorization

\(P(x) - P(a) = (x - a) \cdot Q(x)\)

where \(Q(x) = a(x^2 + ax + a^2) + b(x + a) + c\) is a polynomial of degree 2.

Part 2b: If a is a root, then P(x) = (x - a)Q(x)

Step 1: Start with the given information

We know that a is a root of P, which means P(a) = 0.

Step 2: Use the factorization from part 2a

\(P(x) - P(a) = (x - a) \cdot Q(x)\)

Step 3: Substitute P(a) = 0

\(P(x) - 0 = (x - a) \cdot Q(x)\)
\(P(x) = (x - a) \cdot Q(x)\)

Step 4: Determine the degree of Q

Since P is a cubic polynomial (degree 3) and (x - a) is a linear polynomial (degree 1), Q must be a quadratic polynomial (degree 2).

Step 5: Conclusion

If a is a root of the cubic polynomial P, then P(x) = (x - a)Q(x), where Q is a quadratic polynomial.

Part 2c: Maximum number of roots for a cubic polynomial

Step 1: Understanding the problem

We need to prove that a cubic polynomial (degree 3) can have at most three roots.

Step 2: Use the factorization property

From part 2b, if a is a root of P, then P(x) = (x - a)Q(x), where Q is a quadratic polynomial (degree 2).

Step 3: Consider additional roots

If b is another root of P, then since P(b) = 0 and (b - a) ‚â† 0 (assuming a and b are distinct), we must have Q(b) = 0.

This means b is a root of the quadratic polynomial Q.

Step 4: Continue the factorization

If b is a root of Q, then Q(x) = (x - b)R(x), where R is a linear polynomial (degree 1).

So P(x) = (x - a)(x - b)R(x).

Step 5: Consider a third root

If c is a third root of P, then since P(c) = 0 and (c - a) ‚â† 0, (c - b) ‚â† 0, we must have R(c) = 0.

This means c is a root of the linear polynomial R.

Step 6: Final factorization

If R has a root at c, then R(x) = k(x - c) for some constant k.

So P(x) = k(x - a)(x - b)(x - c).

Step 7: Conclusion

A cubic polynomial can be completely factorized into at most three linear factors, so it can have at most three roots.

Part 3: Quartic Polynomials

Part 3a: Factorization with two distinct roots

Let F be the polynomial defined by

\(F(x) = ax^4 + bx^3 + cx^2 + dx + e\)

where a, b, c, d, e are real numbers with \(a \neq 0\).

Let Œ±, Œ≤ be two distinct roots of F.

Show that there exists a polynomial G such that

\(F(x) = (x - Œ±)(x - Œ≤)G(x)\)

Step 1: Start with the given information

We know that Œ± and Œ≤ are roots of F, so F(Œ±) = 0 and F(Œ≤) = 0.

Step 2: Factor out (x - Œ±)

Since Œ± is a root of F, by the result from part 2b, we can write:

\(F(x) = (x - Œ±)Q(x)\)

where Q is a cubic polynomial (degree 3).

Step 3: Use the second root Œ≤

Since Œ≤ is also a root of F, we have F(Œ≤) = 0.

\(F(Œ≤) = (Œ≤ - Œ±)Q(Œ≤) = 0\)

Step 4: Since Œ± and Œ≤ are distinct

Since Œ± ‚â† Œ≤, we have (Œ≤ - Œ±) ‚â† 0, so we must have Q(Œ≤) = 0.

This means Œ≤ is a root of the cubic polynomial Q.

Step 5: Factor Q(x)

Since Œ≤ is a root of Q, we can write:

\(Q(x) = (x - Œ≤)R(x)\)

where R is a quadratic polynomial (degree 2).

Step 6: Combine the factors

\(F(x) = (x - Œ±)(x - Œ≤)R(x)\)

Step 7: Conclusion

Let G(x) = R(x), which is a quadratic polynomial.

Then F(x) = (x - Œ±)(x - Œ≤)G(x), as required.

Part 3b: Maximum number of roots for a quartic polynomial

Step 1: Understanding the problem

We need to prove that a quartic polynomial (degree 4) can have at most four roots.

Step 2: Use the factorization from part 3a

From part 3a, if Œ± and Œ≤ are roots of F, then F(x) = (x - Œ±)(x - Œ≤)G(x), where G is a quadratic polynomial.

Step 3: Consider additional roots

If Œ≥ is another root of F, then since F(Œ≥) = 0 and (Œ≥ - Œ±) ‚â† 0, (Œ≥ - Œ≤) ‚â† 0, we must have G(Œ≥) = 0.

This means Œ≥ is a root of the quadratic polynomial G.

Step 4: Continue the factorization

If Œ≥ is a root of G, then G(x) = (x - Œ≥)H(x), where H is a linear polynomial (degree 1).

So F(x) = (x - Œ±)(x - Œ≤)(x - Œ≥)H(x).

Step 5: Consider a fourth root

If Œ¥ is a fourth root of F, then since F(Œ¥) = 0 and (Œ¥ - Œ±) ‚â† 0, (Œ¥ - Œ≤) ‚â† 0, (Œ¥ - Œ≥) ‚â† 0, we must have H(Œ¥) = 0.

This means Œ¥ is a root of the linear polynomial H.

Step 6: Final factorization

If H has a root at Œ¥, then H(x) = k(x - Œ¥) for some constant k.

So F(x) = k(x - Œ±)(x - Œ≤)(x - Œ≥)(x - Œ¥).

Step 7: Conclusion

A quartic polynomial can be completely factorized into at most four linear factors, so it can have at most four roots.

Part 4: General Polynomials and Conjectures

Let f be a polynomial of degree n (n > 2) and let Œ±, Œ≤ and Œ≥ be three distinct roots of f.

Part 4a: Conjecture about factorization

Step 1: Analyze the pattern from previous parts

From parts 2 and 3, we observed that:

  • For a cubic polynomial (degree 3) with roots Œ±, Œ≤, Œ≥: \(P(x) = k(x - Œ±)(x - Œ≤)(x - Œ≥)\)
  • For a quartic polynomial (degree 4) with roots Œ±, Œ≤, Œ≥, Œ¥: \(F(x) = k(x - Œ±)(x - Œ≤)(x - Œ≥)(x - Œ¥)\)

Step 2: Formulate the conjecture

Based on this pattern, we can conjecture that:

If a polynomial f of degree n has distinct roots Œ±‚ÇÅ, Œ±‚ÇÇ, ..., Œ±‚Çò, then f can be factorized as:

\(f(x) = k(x - Œ±‚ÇÅ)(x - Œ±‚ÇÇ)\cdots(x - Œ±‚Çò) \cdot g(x)\)

where g(x) is a polynomial of degree (n - m).

Step 3: Specific conjecture for three roots

Specifically, if f has three distinct roots Œ±, Œ≤, Œ≥, then:

\(f(x) = (x - Œ±)(x - Œ≤)(x - Œ≥) \cdot h(x)\)

where h(x) is a polynomial of degree (n - 3).

Part 4b: Conjecture about maximum number of roots

Step 1: Analyze the pattern from previous parts

From parts 2 and 3, we observed that:

  • A cubic polynomial (degree 3) has at most 3 roots
  • A quartic polynomial (degree 4) has at most 4 roots

Step 2: Formulate the general conjecture

Based on this pattern, we can conjecture that:

A polynomial of degree n has at most n roots.

Step 3: Reasoning behind the conjecture

This makes sense because:

  • Each root corresponds to a linear factor (x - r)
  • The product of n linear factors gives a polynomial of degree n
  • If we had more than n roots, we would need more than n linear factors, which would result in a polynomial of degree greater than n

Step 4: Important note

This is known as the Fundamental Theorem of Algebra, which states that a polynomial of degree n has exactly n complex roots (counting multiplicity).

For real roots specifically, a polynomial of degree n has at most n real roots.

Activity 17: Polynomial Factorization Practice

Part 1: Verifying Roots and Factorization

In each of the following cases, verify that a is a root of P and write P(x) in the form \((x - a)Q(x)\), where Q is a polynomial to be determined.

Part 1a: \(P(x) = 2x^3 - 7x^2 - 17x + 10\), a = -2

Step 1: Verify that a = -2 is a root

\(P(-2) = 2(-2)^3 - 7(-2)^2 - 17(-2) + 10\)
\(= 2(-8) - 7(4) + 34 + 10\)
\(= -16 - 28 + 34 + 10 = 0\)

Since P(-2) = 0, -2 is indeed a root of P.

Step 2: Perform polynomial division of P(x) by (x + 2)

\(P(x) = 2x^3 - 7x^2 - 17x + 10\)

Using synthetic division with -2:

Coefficients: 2, -7, -17, 10
Bring down 2: 2
Multiply by -2: -4
Add to -7: -11
Multiply by -2: 22
Add to -17: 5
Multiply by -2: -10
Add to 10: 0

Step 3: Write the factorization

The quotient polynomial has coefficients: 2, -11, 5

\(Q(x) = 2x^2 - 11x + 5\)
\(P(x) = (x + 2)(2x^2 - 11x + 5)\)

Part 1b: \(P(x) = 2x^2 - (1 + \tfrac{2}{3})x - 1 - 13\), a = 1

Step 1: Simplify the polynomial first

\(P(x) = 2x^2 - \left(1 + \frac{2}{3}\right)x - 1 - 13\)
\(= 2x^2 - \frac{5}{3}x - 14\)

Step 2: Verify that a = 1 is a root

\(P(1) = 2(1)^2 - \frac{5}{3}(1) - 14\)
\(= 2 - \frac{5}{3} - 14\)
\(= \frac{6}{3} - \frac{5}{3} - \frac{42}{3} = -\frac{41}{3} \neq 0\)

Since P(1) ‚â† 0, 1 is not a root of P.

Step 3: Check if there might be a typo in the problem

The polynomial might be intended to be: \(P(x) = 2x^2 - (1 + \frac{2}{3})x - 1 - \frac{1}{3}\)

\(= 2x^2 - \frac{5}{3}x - \frac{4}{3}\)

Then P(1) = 2 - 5/3 - 4/3 = 2 - 9/3 = 2 - 3 = -1 ‚â† 0

Since the given value is not a root, we cannot factor as requested.

Part 1c: \(P(x) = x^4 - 3x^3 + x^2 - 4x + 3\), a = 3

Step 1: Verify that a = 3 is a root

\(P(3) = 3^4 - 3(3)^3 + 3^2 - 4(3) + 3\)
\(= 81 - 3(27) + 9 - 12 + 3\)
\(= 81 - 81 + 9 - 12 + 3 = 0\)

Since P(3) = 0, 3 is indeed a root of P.

Step 2: Perform polynomial division of P(x) by (x - 3)

\(P(x) = x^4 - 3x^3 + x^2 - 4x + 3\)

Step 3: Write the factorization

The quotient polynomial has coefficients: 1, 0, 1, -1

\(Q(x) = x^3 + 0x^2 + x - 1 = x^3 + x - 1\)
\(P(x) = (x - 3)(x^3 + x - 1)\)

Part 2: Finding Apparent Roots and Factorization

In each of the following cases, find an apparent root of the polynomial P and factorize P(x).

Part 2a: \(P(x) = x^3 - 11x^2 + 30x\)

Step 1: Look for obvious roots

Notice that P(x) has a common factor of x:

\(P(x) = x(x^2 - 11x + 30)\)

Step 2: Factor the quadratic

\(x^2 - 11x + 30 = (x - 5)(x - 6)\)

Step 3: Complete factorization

\(P(x) = x(x - 5)(x - 6)\)

The roots are 0, 5, and 6.

Part 2b: \(P(x) = x^4 + x^3 - 3x^2 - 4x - 4\)

Step 1: Test possible rational roots

Possible rational roots: ¬±1, ¬±2, ¬±4

\(P(1) = 1 + 1 - 3 - 4 - 4 = -9 \neq 0\)
\(P(-1) = 1 - 1 - 3 + 4 - 4 = -3 \neq 0\)
\(P(2) = 16 + 8 - 12 - 8 - 4 = 0\)

So x = 2 is a root.

Step 2: Factor out (x - 2) using synthetic division

Coefficients: 1, 1, -3, -4, -4
Bring down 1: 1
Multiply by 2: 2
Add to 1: 3
Multiply by 2: 6
Add to -3: 3
Multiply by 2: 6
Add to -4: 2
Multiply by 2: 4
Add to -4: 0

Step 3: Continue with the quotient polynomial

Quotient: \(x^3 + 3x^2 + 3x + 2\)

Test x = -2:

\((-2)^3 + 3(-2)^2 + 3(-2) + 2 = -8 + 12 - 6 + 2 = 0\)

So x = -2 is also a root.

Step 4: Factor out (x + 2)

Coefficients: 1, 3, 3, 2

Quotient: \(x^2 + x + 1\)
\(P(x) = (x - 2)(x + 2)(x^2 + x + 1)\)

Note: \(x^2 + x + 1\) has no real roots (discriminant = 1 - 4 = -3).

Part 2c: \(P(x) = 3x^3 + 4x^2 - 6x - 7\)

Step 1: Test possible rational roots

Possible rational roots: ¬±1, ¬±7, ¬±1/3, ¬±7/3

\(P(1) = 3 + 4 - 6 - 7 = -6 \neq 0\)
\(P(-1) = -3 + 4 + 6 - 7 = 0\)

So x = -1 is a root.

Step 2: Factor out (x + 1) using synthetic division

Coefficients: 3, 4, -6, -7
Bring down 3: 3
Multiply by -1: -3
Add to 4: 1
Multiply by -1: -1
Add to -6: -7
Multiply by -1: 7
Add to -7: 0

Step 3: Complete factorization

Quotient: \(3x^2 + x - 7\)
\(P(x) = (x + 1)(3x^2 + x - 7)\)

The quadratic factor cannot be factored further using integers.

Part 2d: \(P(x) = 2x^2 - (4 + 12)x + 212\)

Step 1: Simplify the polynomial

\(P(x) = 2x^2 - (4 + 12)x + 212\)
\(= 2x^2 - 16x + 212\)

Step 2: Factor out the common factor

\(P(x) = 2(x^2 - 8x + 106)\)

Step 3: Check if the quadratic can be factored

Discriminant = \((-8)^2 - 4(1)(106) = 64 - 424 = -360\)

Since the discriminant is negative, the quadratic has no real roots and cannot be factored over the reals.

Step 4: Complete factorization

\(P(x) = 2(x^2 - 8x + 106)\)

This is the complete factorization over the real numbers.
