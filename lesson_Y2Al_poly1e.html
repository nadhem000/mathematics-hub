Key Concepts Summary

Equality of Two Polynomials

Zero Polynomial

A polynomial is identically zero if and only if all its coefficients are zero.

\(P(x) = 0 \text{ for all } x \iff a_n = a_{n-1} = \cdots = a_1 = a_0 = 0\)

Polynomial Equality

Two polynomials are equal if and only if they have the same degree and the coefficients of the same degree terms are equal.

\(P(x) = Q(x) \text{ for all } x \iff \deg(P) = \deg(Q) \text{ and corresponding coefficients are equal}\)

Operations on Polynomial Functions

Operations Theorem

If \(f\) and \(g\) are two non-zero polynomials and \(Î±\) is a non-zero real number then:

  • \(f + g\) is a polynomial
  • \(f \times g\) is a polynomial and \(\deg(f \times g) = \deg(f) + \deg(g)\)
  • \(Î±f\) is a polynomial and \(\deg(Î±f) = \deg(f)\)

Roots of a Polynomial

Fundamental Theorem on Roots

A polynomial of degree \(n\) has at most \(n\) real roots.

\(\deg(P) = n \Rightarrow P \text{ has at most } n \text{ roots}\)

- A polynomial of degree 3 has at most three roots

- A polynomial of degree 4 has at most four roots

Factorization of a Polynomial

Factorization Theorem

Let \(f\) be a polynomial of degree \(n\).

Single Root Case

For \(n \geq 1\), if \(Î±\) is a root of \(f\) then:

  • \(f\) is factorable by \((x - Î±)\)
  • There exists a polynomial \(g\) of degree \((n - 1)\) such that \(f(x) = (x - Î±)g(x)\)
Two Roots Case

For \(n \geq 2\), if \(Î±\) and \(Î²\) are roots of \(f\) then:

  • \(f\) is factorable by \((x - Î±)(x - Î²)\)
  • There exists a polynomial \(g\) of degree \((n - 2)\) such that \(f(x) = (x - Î±)(x - Î²)g(x)\)
General Case

Let \(f\) be a polynomial of degree \(n\) (\(n \geq 3\)). If \(Î±_1, Î±_2, Î±_3, \ldots, Î±_k\) (with \(k \leq n\)) are roots of \(f\) then:

  • \(f\) is factorable by \((x - Î±_1)(x - Î±_2)(x - Î±_3)\ldots(x - Î±_k)\)
  • There exists a polynomial \(g\) of degree \((n - k)\) such that:
\(f(x) = (x - Î±_1)(x - Î±_2)(x - Î±_3)\ldots(x - Î±_k)g(x)\)

Exercise 1: Polynomial Operations

Let \(f(x) = x^4 + x^2 - 5x + 1\), \(g(x) = 5x^3 + x + 1\), and \(h(x) = 2x^2 - 7x - 2\).

Reduce and order the following expressions:

Part a: \(f(x) - h(x)\)

Step 1: Write the expression

\(f(x) - h(x) = (x^4 + x^2 - 5x + 1) - (2x^2 - 7x - 2)\)

Step 2: Distribute the negative sign

\(= x^4 + x^2 - 5x + 1 - 2x^2 + 7x + 2\)

Step 3: Combine like terms

\(= x^4 + (x^2 - 2x^2) + (-5x + 7x) + (1 + 2)\)
\(= x^4 - x^2 + 2x + 3\)

Step 4: Final answer

\(f(x) - h(x) = x^4 - x^2 + 2x + 3\)

Part b: \(2g(x) - 5h(x)\)

Step 1: Write the expression

\(2g(x) - 5h(x) = 2(5x^3 + x + 1) - 5(2x^2 - 7x - 2)\)

Step 2: Distribute the coefficients

\(= 10x^3 + 2x + 2 - 10x^2 + 35x + 10\)

Step 3: Combine like terms

\(= 10x^3 - 10x^2 + (2x + 35x) + (2 + 10)\)
\(= 10x^3 - 10x^2 + 37x + 12\)

Step 4: Final answer

\(2g(x) - 5h(x) = 10x^3 - 10x^2 + 37x + 12\)

Exercise 2: Verifying Roots

Let \(f(x) = x^3 + x^2 - 8x - 6\).

Verify that \(1 - \sqrt{3}\) is a root of \(f\).

Step 1: Substitute \(x = 1 - \sqrt{3}\) into \(f(x)\)

\(f(1 - \sqrt{3}) = (1 - \sqrt{3})^3 + (1 - \sqrt{3})^2 - 8(1 - \sqrt{3}) - 6\)

Step 2: Calculate \((1 - \sqrt{3})^2\)

\((1 - \sqrt{3})^2 = 1^2 - 2(1)(\sqrt{3}) + (\sqrt{3})^2 = 1 - 2\sqrt{3} + 3 = 4 - 2\sqrt{3}\)

Step 3: Calculate \((1 - \sqrt{3})^3\)

\((1 - \sqrt{3})^3 = (1 - \sqrt{3})(1 - \sqrt{3})^2 = (1 - \sqrt{3})(4 - 2\sqrt{3})\)
\(= 1(4 - 2\sqrt{3}) - \sqrt{3}(4 - 2\sqrt{3}) = 4 - 2\sqrt{3} - 4\sqrt{3} + 2(\sqrt{3})^2\)
\(= 4 - 6\sqrt{3} + 2(3) = 4 - 6\sqrt{3} + 6 = 10 - 6\sqrt{3}\)

Step 4: Substitute all values

\(f(1 - \sqrt{3}) = (10 - 6\sqrt{3}) + (4 - 2\sqrt{3}) - 8(1 - \sqrt{3}) - 6\)
\(= 10 - 6\sqrt{3} + 4 - 2\sqrt{3} - 8 + 8\sqrt{3} - 6\)

Step 5: Combine like terms

\(= (10 + 4 - 8 - 6) + (-6\sqrt{3} - 2\sqrt{3} + 8\sqrt{3})\)
\(= 0 + 0 = 0\)

Step 6: Conclusion

Since \(f(1 - \sqrt{3}) = 0\), \(1 - \sqrt{3}\) is indeed a root of \(f(x)\).

Exercise 10: Polynomial with Parameter

Let \(a\) be a real number and let \(f\) be the polynomial defined by:

\(f(x) = (x - a + 2)^3 - (x^3 - a^3 + 8)\)

Verify that \(a\) and \(-2\) are roots of \(f\).

Step 1: Verify that \(a\) is a root

\(f(a) = (a - a + 2)^3 - (a^3 - a^3 + 8) = (2)^3 - (0 + 8) = 8 - 8 = 0\)

Step 2: Verify that \(-2\) is a root

\(f(-2) = (-2 - a + 2)^3 - ((-2)^3 - a^3 + 8) = (-a)^3 - (-8 - a^3 + 8)\)
\(= -a^3 - (-a^3) = -a^3 + a^3 = 0\)

Step 3: Conclusion

Both \(f(a) = 0\) and \(f(-2) = 0\), so \(a\) and \(-2\) are roots of \(f(x)\).

Exercise 11: Finding and Using Roots

Let \(P(x) = x^3 + 6x^2 + 12x - 56\).

Part a: Find an integer root of \(P\)

Step 1: Use the Rational Root Theorem

Possible rational roots are factors of 56: \(\pm 1, \pm 2, \pm 4, \pm 7, \pm 8, \pm 14, \pm 28, \pm 56\)

Step 2: Test possible roots

\(P(2) = (2)^3 + 6(2)^2 + 12(2) - 56 = 8 + 24 + 24 - 56 = 0\)

Step 3: Conclusion

Since \(P(2) = 0\), \(x = 2\) is an integer root of \(P(x)\).

Part b: Solve the equation \(P(x) = 0\)

Step 1: Use the root found in part a

Since \(x = 2\) is a root, we can factor \(P(x)\) as \((x - 2)Q(x)\)

Step 2: Perform polynomial division

\(P(x) = (x - 2)(x^2 + 8x + 28)\)

Step 3: Solve the quadratic equation

\(x^2 + 8x + 28 = 0\)
\(\text{Discriminant: } \Delta = 8^2 - 4(1)(28) = 64 - 112 = -48\)
\(x = \frac{-8 \pm \sqrt{-48}}{2} = \frac{-8 \pm 4i\sqrt{3}}{2} = -4 \pm 2i\sqrt{3}\)

Step 4: Final solutions

\(x = 2, \quad x = -4 + 2i\sqrt{3}, \quad x = -4 - 2i\sqrt{3}\)
