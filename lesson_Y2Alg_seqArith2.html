Arithmetic Sequences: Part II

Definition, General Term, Graphical Representation, and Sum of Terms

Introduction

In this lesson, we will continue our study of arithmetic sequences. We will learn how to recognize arithmetic sequences, find their general term, represent them graphically, and calculate the sum of consecutive terms.

Arithmetic sequences are everywhere in real life! Think about: saving money regularly, the height of a staircase, or the distance traveled by a car at constant speed. Understanding arithmetic sequences helps us solve many practical problems.

Tip for Beginners

An arithmetic sequence is like climbing stairs where each step has the same height. The "reason" (ratio) tells you how much you go up (or down) each time. Don't worry if it seems difficult at first - we'll go step by step!

Definition

A sequence (Un) is called arithmetic if there exists a real number r such that, for every natural number n, we have:

Un+1 = Un + r

The real number r is called the common difference or reason of the arithmetic sequence.

Remark:

If a, b, c, d,... are consecutive terms of an arithmetic sequence, we say they are in arithmetic progression.

Understanding the Definition

Think of an arithmetic sequence like a staircase where each step has the same height. The "r" (common difference) is the height of each step. If r is positive, we go up; if r is negative, we go down; if r is zero, we stay at the same level.

In Activity 1, we found that Un+1 - Un = 5 for all n. This means r = 5, so (Un) is an arithmetic sequence with common difference 5.

1) a) Let S = 1 + 2 + ... + 99 + 100.

By writing S = 100 + 99 + ... + 2 + 1, calculate 2S and deduce S.

Step 1: Writing S in two different orders

S = 1 + 2 + 3 + ... + 98 + 99 + 100

S = 100 + 99 + 98 + ... + 3 + 2 + 1

Step 2: Adding the two expressions for S

When we add them term by term, we get:

S + S = (1+100) + (2+99) + (3+98) + ... + (98+3) + (99+2) + (100+1)

2S = 101 + 101 + 101 + ... + 101 + 101 + 101

Step 3: Counting how many terms

How many 101's do we have? We started with 100 terms in S, so we have 100 pairs, each summing to 101.

2S = 101 Ã— 100

2S = 10,100

Step 4: Solving for S

S = 10,100 Ã· 2 = 5,050

Step 5: Final answer

S = 1 + 2 + ... + 99 + 100 = 5,050

Step 6: Historical note

This method is said to have been discovered by the mathematician Carl Friedrich Gauss when he was a child. His teacher asked the class to add all numbers from 1 to 100, and Gauss found the answer quickly using this trick!

b) Calculate 500 + 501 + ... + 1999 + 2000.

Step 1: Understanding what we need to calculate

We need to calculate: 500 + 501 + 502 + ... + 1999 + 2000

Let S = 500 + 501 + ... + 1999 + 2000

Write S in reverse order: S = 2000 + 1999 + ... + 501 + 500

Add the two expressions:

2S = (500+2000) + (501+1999) + (502+1998) + ... + (1999+501) + (2000+500)

2) Let S be the sum of n consecutive terms of an arithmetic sequence with common difference r. Let a be the first term and b be the last term of this sum.

By arranging the terms of this sum in two ways:

  • From first to last: S = a + (a + r) + (a + 2r) + ... + [a + (n - 1)r]
  • From last to first: S = b + (b - r) + (b - 2r) + ... + [b - (n - 1)r]

Step 1: Writing S in two different orders

S = a + (a + r) + (a + 2r) + ... + [a + (n-2)r] + [a + (n-1)r]

The last term b = a + (n-1)r

S = b + (b - r) + (b - 2r) + ... + [b - (n-2)r] + [b - (n-1)r]

Step 2: Adding the two expressions for S

When we add them term by term:

S + S = [a + b] + [(a+r) + (b-r)] + [(a+2r) + (b-2r)] + ... + [(a+(n-1)r) + (b-(n-1)r)]

Step 3: Simplifying each pair

Each pair simplifies to (a + b):

  • First pair: a + b
  • Second pair: (a+r) + (b-r) = a + b
  • Third pair: (a+2r) + (b-2r) = a + b
  • Last pair: [a+(n-1)r] + [b-(n-1)r] = a + b

Step 4: Counting the pairs

We have n terms in the sum, so we have n pairs, each equal to (a + b).

2S = n Ã— (a + b)

Step 5: Solving for S

S = n Ã— (a + b) Ã· 2

Step 6: Final formula

The sum of n consecutive terms of an arithmetic sequence is:

S = n Ã— (first term + last term) Ã· 2

Or in symbols: S = n Ã— (a + b) / 2

Summary of Key Concepts

1. Arithmetic Sequence Definition

A sequence (Un) is arithmetic if there exists a real number r (called the common difference) such that:

Un+1 = Un + r for all n

Example: 3, 8, 13, 18, 23,... (r = 5)

2. General Term Formula

If (Un) is arithmetic with first term U0 and common difference r, then:

4. Sum of Consecutive Terms
