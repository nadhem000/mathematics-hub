3) Consider the sequence (tn) defined by: t0 = 2 and tn+1 = tn - 3, n ‚àà ‚Ñï.

Step 1: Understanding the definition

We know: t0 = 2 (the first term)

And: tn+1 = tn - 3

This means: "To get the next term, take the current term and subtract 3"

What does tn+1 = tn - 3 mean?

This is called a recursive definition. It tells us how to find each term from the previous one:

  • t1 = t0 - 3
  • t2 = t1 - 3
  • t3 = t2 - 3
  • And so on...

Step 2: Calculating t1

t1 = t0 - 3 = 2 - 3 = -1

Step 3: Calculating t2

t2 = t1 - 3 = (-1) - 3 = -4

Step 4: Calculating t3

t3 = t2 - 3 = (-4) - 3 = -7

Step 5: Calculating t4

t4 = t3 - 3 = (-7) - 3 = -10

Step 6: Calculating t5

t5 = t4 - 3 = (-10) - 3 = -13

Step 7: Final answers

  • t0 = 2
  • t1 = -1
  • t2 = -4
  • t3 = -7
  • t4 = -10
  • t5 = -13

Remark: Notice that each term is 3 less than the previous term. This sequence is actually an arithmetic sequence with common difference r = -3.

What is a Sequence?

A sequence is a list of numbers in a specific order. Each number in the list is called a term.

Example: 2, 5, 8, 11, 14,... is a sequence

There are two main ways to define a sequence:

  • Explicit formula: Un = 2n + n (tells you how to calculate any term directly)
  • Recursive formula: U0 = 2, Un+1 = Un - 3 (tells you how to get each term from the previous one)

Show that each of the following sequences is arithmetic, and specify in each case the first term and the common difference.

How to check if a sequence is arithmetic:

A sequence is arithmetic if the difference between consecutive terms is constant. We check by calculating Un+1 - Un and seeing if it's always the same number.

a) The sequence (Un) defined by Un = (-2n + 1)/8, n ‚àà ‚Ñï.

Step 1: Write the general formula

Un = (-2n + 1)/8

Step 2: Calculate Un+1

To find Un+1, replace n with (n+1) in the formula:

Un+1 = [-2(n+1) + 1]/8 = (-2n - 2 + 1)/8 = (-2n - 1)/8

Step 3: Calculate Un+1 - Un

Un+1 - Un = [(-2n - 1)/8] - [(-2n + 1)/8]

Un+1 - Un = (-2n - 1 + 2n - 1)/8

Un+1 - Un = (-2)/8 = -1/4

Step 4: Conclusion about the sequence

Since Un+1 - Un = -1/4 for all n, the difference is constant.

Therefore, (Un) is an arithmetic sequence.

Step 5: Find the first term

The first term is U0 (when n = 0):

U0 = (-2√ó0 + 1)/8 = 1/8

Step 6: Identify the common difference

From Step 3, we found that Un+1 - Un = -1/4

So the common difference is r = -1/4

Step 7: Final answer

(Un) is an arithmetic sequence with:

  • First term: U0 = 1/8
  • Common difference: r = -1/4

b) The sequence (Un) defined by Un = (n/5) - 4, n ‚àà ‚Ñï.

Step 1: Write the general formula

Un = (n/5) - 4

Let's rewrite the formula:

Un = (1/5)n - 4

This is easier to work with because it shows n multiplied by 1/5.

Step 2: Calculate Un+1

To find Un+1, replace n with (n+1) in the formula:

Un+1 = [(n+1)/5] - 4 = (n/5 + 1/5) - 4 = (n/5) - 4 + 1/5

Step 3: Calculate Un+1 - Un

Un+1 - Un = [(n/5) - 4 + 1/5] - [(n/5) - 4]

Un+1 - Un = (n/5) - 4 + 1/5 - (n/5) + 4

Un+1 - Un = 1/5

Step 4: Conclusion about the sequence

Since Un+1 - Un = 1/5 for all n, the difference is constant.

Therefore, (Un) is an arithmetic sequence.

Step 5: Find the first term

The first term is U0 (when n = 0):

U0 = (0/5) - 4 = -4

Step 6: Identify the common difference

From Step 3, we found that Un+1 - Un = 1/5

So the common difference is r = 1/5

Step 7: Verify with a few terms

Let's check a few terms to make sure:

  • U0 = -4
  • U1 = (1/5) - 4 = -3.8 or -19/5
  • U2 = (2/5) - 4 = -3.6 or -18/5
  • Check: U1 - U0 = -3.8 - (-4) = 0.2 = 1/5 ‚úì
  • Check: U2 - U1 = -3.6 - (-3.8) = 0.2 = 1/5 ‚úì

Step 8: Final answer

(Un) is an arithmetic sequence with:

  • First term: U0 = -4
  • Common difference: r = 1/5
Quick Way to Recognize Arithmetic Sequences

If a sequence is defined by Un = a √ó n + b (where a and b are constants), then it's arithmetic!

Why? Because:

  • Un+1 = a √ó (n+1) + b = a √ó n + a + b
  • Un+1 - Un = (a √ó n + a + b) - (a √ó n + b) = a
  • So the common difference r = a

Examples from Activity 2:

  • Un = (-2n + 1)/8 = (-1/4)n + 1/8 ‚Üí r = -1/4
  • Un = (n/5) - 4 = (1/5)n - 4 ‚Üí r = 1/5

1) Calculate the common difference of an arithmetic sequence knowing that U15 = -12 and U24 = 15.

Step 1: Recall the formula for arithmetic sequences

For an arithmetic sequence: Un = U0 + n √ó r

Where U0 is the first term and r is the common difference.

Important Note:

If the sequence doesn't start at n=0, we can still use a similar formula. For example, if we know two terms Ua and Ub, we can write:

Ub = Ua + (b - a) √ó r

Step 2: Apply the formula to our problem

We know U15 = -12 and U24 = 15

Think of U24 as coming from U15:

U24 = U15 + (24 - 15) √ó r

15 = -12 + 9 √ó r

Step 3: Solve for r

15 = -12 + 9r

Add 12 to both sides: 15 + 12 = 9r

27 = 9r

Divide both sides by 9:

r = 27 √∑ 9 = 3

Step 4: Verify our answer

If r = 3, then going from U15 to U24 (9 steps) should add 9 √ó 3 = 27

U15 = -12, so U24 should be -12 + 27 = 15 ‚úì

Step 5: Final answer

The common difference is r = 3

2) (Un) is an arithmetic sequence with common difference r = 0.5 and such that U30 = 11/2. Determine U20 and U50.

Step 1: Write down what we know

  • Common difference: r = 0.5 = 1/2
  • U30 = 11/2 = 5.5

Step 2: Find U20

To go from U30 to U20, we go backward 10 steps (30 - 20 = 10)

Since we're going backward, we subtract: 10 √ó r = 10 √ó 0.5 = 5

U20 = U30 - (30 - 20) √ó r = 5.5 - 10 √ó 0.5

U20 = 5.5 - 5 = 0.5 = 1/2

Step 3: Find U50

To go from U30 to U50, we go forward 20 steps (50 - 30 = 20)

Since we're going forward, we add: 20 √ó r = 20 √ó 0.5 = 10

U50 = U30 + (50 - 30) √ó r = 5.5 + 20 √ó 0.5

U50 = 5.5 + 10 = 15.5 = 31/2

Step 4: Alternative method (using the general formula)

We know: Un = U0 + n √ó r

First, find U0 using U30 = 5.5:

5.5 = U0 + 30 √ó 0.5 = U0 + 15

U0 = 5.5 - 15 = -9.5

Now find U20:

U20 = -9.5 + 20 √ó 0.5 = -9.5 + 10 = 0.5 ‚úì

Now find U50:

U50 = -9.5 + 50 √ó 0.5 = -9.5 + 25 = 15.5 ‚úì

Step 5: Final answers

  • U20 = 1/2 = 0.5
  • U50 = 31/2 = 15.5

Important Formula to Remember

For an arithmetic sequence, the sum of n terms is:

S = n √ó (first term + last term) √∑ 2
